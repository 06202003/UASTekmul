Frequently Asked Question

Bootstrap adalah framework HTML, CSS, dan JavaScript yang berfungsi untuk mendesain website responsive dengan cepat dan mudah. Framework open source ini diciptakan pada tahun 2011 oleh Mark Otto dan Jacob Thornton dari Twitter. Itulah kenapa dulunya Bootstrap dinamakan Twitter Blueprint.

Kegunaan Bootstrap dalam pengembangan website

  • Menciptakan website Mobile Friendly
  • Memudahkan resize gambar
  • Menambahkan elemen website tanpa ribet
  • Membuat website lebih interaktif

Beberapa kelebihan Bootstrap yang membuatnya menjadi salah satu framework terpopuler di dunia adalah:

  • Ramah untuk pemula
  • Grid System yang canggih
  • Kompabilitas dengan web browser baru
  • Bersifat open source
  • Kebebasan kostumisasi
  • Rutin diperbarui
  • Tersedia dokumentasi lengkap
  • Memiliki komunitas aktif yang lengkap
